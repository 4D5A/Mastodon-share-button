Mastodon Share Button

Share something by sending data...

Mastodon Share Button is a little library to add buttons for sharing a web page on Mastodon.
At your right, the button works by sending data to library button by button. For example, there is the source code to describe a button with data:

<div class="mastodon-share-button" data-target="https://framapiaf.org" data-name="Framapiaf" data-buttonstyle="btn btn-primary" ></div>

Framapiaf.org

Framapiaf est un service de microblog similaire à Twitter. Il est libre, décentralisé et fédéré. Il permet de courts messages (max. 500 caractères), de définir leur degré de confidentialité et de suivre les membres du réseau sans publicité ni pistage.
